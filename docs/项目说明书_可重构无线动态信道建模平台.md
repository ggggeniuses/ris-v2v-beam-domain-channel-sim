# 可重构无线动态信道建模平台

## 项目定位

本项目是一个 MATLAB 研究型仿真平台，用于分析 RIS 和 FAS 可重构无线
系统中的动态信道、传播统计、波束域稀疏性、建模复杂度与信道容量。

当前包含两个主要场景：

1. RIS-V2V：RIS 辅助车联网信道的空间、时间和频率相关特性。
2. FAS-UAV：流体天线辅助无人机对地通信中的端口选择和容量分析。

项目重点是形成结构清晰、可运行、可重复生成结果的科研工程，而不是提出
一个未经验证的新信道模型。

## 已实现功能

### RIS-V2V

- Spatial CCF、Temporal ACF 和 Frequency FCF。
- RIS 阵列尺寸和单元间距扫描。
- 不同车辆运动状态下的相关性分析。
- 阵列域与波束域的理想容量不变性检查。
- BDCM 能量集中度和稀疏性指标。
- 完整模型与 95% 能量稀疏模型的同口径复杂度比较。

### FAS-UAV

- 动态 LoS/NLoS 信道生成。
- 流体天线有效端口选择。
- 端口间距和有效端口数量对应的建模误差。
- 不同天线孔径和 UAV 运动状态下的容量。
- FAS 与传统 ULA 的容量比较。
- `quick`、`calibration` 和 `final` 三种 Monte Carlo 配置。

## 运行方式

```matlab
run_all
run_tests
```

分别运行：

```matlab
run_benchmarks
run_extensions
run_fas_uav
```

最终配置：

```powershell
$env:FAS_UAV_PROFILE = "final"
matlab -batch "run_all"
matlab -batch "run_tests"
```

## 验证范围

- DFT 波束形成矩阵酉性。
- 阵列域与波束域理想容量一致性。
- 固定随机种子下的信道确定性。
- 有效端口索引和信道矩阵尺寸。
- 原始 CIR 与容量归一化信道的独立执行路径。
- 建模误差公式。
- 稀疏性和复杂度指标。
- 输出文件完整性与有限数值检查。
